# 가이드라인생성 스킬

## 사용법

```
/가이드라인생성 파일=[파일경로] 양식명=[가이드라인 이름] (기관=[기관명])
```

- `파일`, `양식명`: 필수
- `기관`: 선택. 지정 시 `docs/guidelines/[기관명]/` 하위에 저장. 생략 시 정부 표준으로 간주하여 `docs/guidelines/` 루트에 저장.

**예시**
```
/가이드라인생성 파일=samples/협조문_예시.docx 양식명=협조문
/가이드라인생성 파일=samples/서부발전_기안문.hwp 양식명=기안문 기관=한국서부발전
/가이드라인생성 파일=samples/내부결재_예시.pdf 양식명=내부결재문 기관=한국전력공사
```

---

## 실행 절차

### 1단계: 입력값 파싱 및 컨텍스트 조립

```
CONTEXT:
  파일: [파일경로]
  양식명: [양식명]
  기관: [기관명 | 없음]
```

`파일` 또는 `양식명`이 누락된 경우 사용자에게 입력을 요청합니다.

### 2단계: 저장 경로 및 중복 확인

`기관` 값에 따라 저장 경로를 결정합니다.

- `기관`이 있는 경우: `docs/guidelines/[기관명]/[양식명]_가이드라인.md`
- `기관`이 없는 경우: `docs/guidelines/[양식명]_가이드라인.md`

해당 경로에 파일이 이미 존재하면 사용자에게 덮어쓸지 확인합니다.
덮어쓰지 않으면 작업을 종료합니다.

### 3단계: 파일 분석 (doc-analyzer 서브에이전트 호출)

`doc-analyzer` 서브에이전트에 다음 컨텍스트를 전달합니다.
- `FILE_PATH`: 파일 경로
- `FORMAT_NAME`: 양식명
- `기관`: 기관명 (있을 경우)

서브에이전트는 파일을 읽고 구조·규칙·문체 패턴을 분석하여 반환합니다.

### 4단계: 가이드라인 파일 생성

doc-analyzer의 분석 결과를 바탕으로 가이드라인 파일을 Write 도구로 생성합니다.

저장 경로:
- 정부 표준: `docs/guidelines/[양식명]_가이드라인.md`
- 기관별: `docs/guidelines/[기관명]/[양식명]_가이드라인.md`

가이드라인 파일은 `기안문_가이드라인.md`와 동일한 구조를 따릅니다.
- 1. 정의 및 용도
- 2. 문서 구성 요소
- 3. 항목 번호 체계
- 4. 작성 규칙
- 5. 마크다운 출력 형식
- 6. 자주 하는 실수
- 7. 작성 체크리스트

`기관`이 지정된 경우, 가이드라인 상단에 다음을 추가합니다.
```
> 이 가이드라인은 [기관명] 전용입니다.
> 정부 표준과 다른 규칙은 ⚠️ 표시로 구분합니다.
```

기관별 가이드라인 폴더가 없으면 Bash 도구로 생성합니다.
```bash
mkdir -p "docs/guidelines/[기관명]"
mkdir -p "docs/examples/[기관명]/[양식명]"
```

### 5단계: 결과 안내

```
가이드라인 생성 완료
저장 위치: docs/guidelines/[경로]/[양식명]_가이드라인.md

[분석된 주요 특징 3~5줄 요약]

이제 다음 명령으로 문서를 작성할 수 있습니다.
/공문작성 양식=[양식명] 주제=[주제] 기관=[기관명]
```

## 지원 파일 형식 및 처리 방식

| 형식 | 처리 방법 | 비고 |
|------|-----------|------|
| `.docx` | pandoc 또는 python-docx로 텍스트 추출 | 가장 안정적 |
| `.pdf` | pdfplumber 또는 pdfminer로 텍스트 추출 | 레이아웃 정보 일부 손실 가능 |
| `.hwp` | pyhwp(hwp5txt)로 텍스트 추출 | 미설치 시 LibreOffice 변환 시도 |
| `.txt` / `.md` | Read 도구로 직접 읽기 | 변환 불필요 |

변환 도구가 없어 파일을 읽지 못하는 경우 사용자에게 안내합니다.
- DOCX 또는 PDF로 변환 후 재시도
- 또는 문서 내용을 텍스트로 직접 붙여넣기 후 재호출

---

## 오류 처리

| 상황 | 대응 |
|------|------|
| 파일이 존재하지 않음 | 경로 재확인 요청 |
| 지원하지 않는 파일 형식 | 지원 형식 안내 및 변환 요청 |
| 파일 읽기 실패 (도구 미설치) | 변환 방법 또는 텍스트 붙여넣기 안내 |
| 같은 이름 가이드라인 존재 | 덮어쓰기 여부 확인 후 처리 |
| 분석 결과 불충분 | 추가 예시 파일 요청 또는 수동 보완 안내 |
