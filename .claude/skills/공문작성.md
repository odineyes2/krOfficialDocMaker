# 공문작성 스킬

## 사용법

```
/공문작성 양식=[양식명] 주제=[주제] (기관=[기관명]) (분량=[페이지수])
```

- `양식`, `주제`: 필수
- `기관`: 선택. 특정 공기업·공단의 양식을 사용할 경우 입력. 생략 시 정부 표준 양식 적용.
- `분량`: 선택. 목표 페이지 수(정수). 생략 시 주제와 양식에 따라 자동 결정.

**예시**
```
/공문작성 양식=기안문 주제=2026년도 하반기 예산 지원 요청
/공문작성 양식=기안문 주제=재난대응훈련 계획(안) 기관=한국서부발전 분량=2
/공문작성 양식=협조문 주제=업무협약 체결 협조 요청 기관=한국전력공사
```

---

## 실행 절차

### 1단계: 입력값 파싱 및 컨텍스트 조립

사용자 입력을 파싱하여 다음 컨텍스트 블록을 조립합니다.

```
CONTEXT:
  양식: [양식명]
  주제: [주제]
  기관: [기관명 | 없음]
  분량: [페이지수 | 자동]
```

- `양식` 또는 `주제`가 누락된 경우 사용자에게 입력을 요청합니다.
- `기관`이 없으면 `없음`으로 설정합니다.
- `분량`이 없으면 `자동`으로 설정합니다.

### 2단계: 양식 선택 (format-selector 서브에이전트 호출)

`format-selector` 서브에이전트에 위 컨텍스트 블록을 전달합니다.

서브에이전트는 다음 우선순위로 가이드라인 파일을 탐색합니다.
1. `기관`이 지정된 경우 → `docs/guidelines/[기관명]/[양식명]_가이드라인.md` 우선 탐색
2. 없으면 → `docs/guidelines/[양식명]_가이드라인.md` (정부 표준) 폴백

반환값이 `NOT_FOUND`인 경우 다음 절차를 따릅니다.
1. 요청한 양식이 아직 라이브러리에 없음을 안내합니다.
2. 현재 사용 가능한 양식 목록을 제시합니다.
3. 다음 선택지를 제안합니다.
   - **가**: 가장 유사한 기존 양식으로 진행
   - **나**: `/가이드라인생성`으로 새 가이드라인 등록 후 재시도
   - **다**: 작업 취소
4. 사용자의 선택을 기다립니다.

### 3단계: 문서 초안 작성 (doc-writer 서브에이전트 호출)

`doc-writer` 서브에이전트에 다음을 전달합니다.
- 컨텍스트 블록 (전체)
- 가이드라인 파일 경로

서브에이전트는 가이드라인을 읽고 컨텍스트의 `분량`에 맞춰 마크다운 초안을 작성합니다.

### 4단계: 문서 검토 (doc-reviewer 서브에이전트 호출)

`doc-reviewer` 서브에이전트에 다음을 전달합니다.
- 가이드라인 파일 경로
- doc-writer가 작성한 초안

서브에이전트는 가이드라인 기준으로 초안을 검토하고 교정된 최종본을 반환합니다.

### 5단계: 파일 저장

최종 공문을 `output/` 디렉토리에 마크다운 파일로 저장합니다.

- 파일명 형식: `YYYYMMDD_[기관명_][양식명]_[주제 요약].md`
  - `기관`이 있는 경우: `20260626_한국서부발전_기안문_재난대응훈련_계획안.md`
  - `기관`이 없는 경우: `20260626_기안문_재난대응훈련_계획안.md`
- `output/` 디렉토리가 없으면 Bash 도구로 생성합니다.
- Write 도구로 파일을 저장합니다.

### 6단계: 결과 출력

최종 공문을 사용자에게 출력하고 저장 경로를 안내합니다.

```
---
[기관명] [양식명] 작성 완료
---

[최종 공문 내용]

---
※ 위 공문은 [양식명] 가이드라인을 기반으로 작성되었습니다.
   [ ] 표시 항목은 실제 기관 정보로 수정 후 사용하시기 바랍니다.

저장 위치: output/[파일명].md
```

## 오류 처리

| 상황 | 대응 |
|------|------|
| 양식명 누락 | 사용자에게 입력 요청 |
| 주제 누락 | 사용자에게 입력 요청 |
| 양식 NOT_FOUND | 목록 안내 + 3가지 선택지 제시 |
| 기관 폴더 없음, 표준 있음 | 표준 가이드라인으로 자동 폴백 후 안내 |
| output/ 디렉토리 없음 | Bash로 자동 생성 후 저장 진행 |
| 파일 저장 실패 | 사용자에게 오류 안내 후 화면 출력만 제공 |
