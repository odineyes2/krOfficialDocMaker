# 기안문 작성 가이드라인

## 1. 정의 및 용도

기안문은 공문서 작성의 시작 단계로, 담당자(기안자)가 업무 처리 내용을 작성하여 결재를 요청하는 문서입니다.
결재 완료 후 외부 발신 시 **시행문**이 되며, 내부 처리로 종결될 경우 **내부결재문**이 됩니다.

**적용 상황**
- 외부 기관·단체·개인에게 공식 의사를 전달할 때
- 내부적으로 결재가 필요한 사항을 처리할 때
- 법령·지침·지시 등에 따른 업무를 시행할 때

---

## 2. 문서 구성 요소

기안문은 **두문 → 본문 → 결문** 3단계로 구성됩니다.

### 2-1. 두문 (頭文)

| 항목 | 설명 | 필수 여부 |
|------|------|-----------|
| 수신 | 문서를 받는 기관·부서·개인 | 필수 (내부결재 시 생략) |
| 경유 | 중간 경유 기관 | 해당 시 기재 |

- 수신자가 여러 명일 경우: `수신: 수신처 참조` 후 결문 하단에 수신처 목록 기재
- 내부결재문: 수신·경유 항목 없이 시작

### 2-2. 본문 (本文)

| 항목 | 설명 |
|------|------|
| 제목 | 문서 내용을 압축한 명사형 제목 (20자 내외 권장) |
| 내용 | 목적·경위·주요내용·협조사항 등을 항목 번호로 구분 |
| 붙임 | 첨부 파일 목록 (없으면 생략) |

### 2-3. 결문 (結文)

| 항목 | 설명 |
|------|------|
| 발신기관명 | 문서를 발신하는 기관의 공식 명칭 |
| 대표자 직위·성명 | 기관장 또는 위임받은 결재권자 |
| 담당자 정보 | 기안자 부서·직위·성명·연락처 |
| 공개구분 | 공개 / 부분공개 / 비공개 중 선택 |

---

## 3. 항목 번호 체계

본문 내용의 항목은 다음 순서를 따릅니다.

```
1.  (대항목)
  가.  (중항목)
    1)  (소항목)
      가)  (세부항목)
        (1)  (더 세부)
          (가)  (최세부)
```

- 항목이 하나뿐일 경우에도 번호를 붙임
- 각 항목 내용이 두 줄 이상이면 항목 번호 아래에서 들여쓰기

---

## 4. 작성 규칙

### 4-1. 숫자·날짜·금액

| 항목 | 규칙 | 예시 |
|------|------|------|
| 날짜 | 년. 월. 일. (마침표 포함) | `2024. 6. 26.` |
| 숫자 | 아라비아 숫자 | `15명`, `3건` |
| 금액 | 숫자 + 괄호 한글 | `1,000,000원(일백만원)` |
| 기간 | 물결표로 연결 | `2024. 1. 1. ~ 2024. 12. 31.` |

### 4-2. 문체

- **경어체** 사용: `~합니다`, `~하겠습니다`, `~드립니다`
- **간결·명확**: 하나의 문장은 가능한 짧게
- **능동형** 선호: `~을 요청합니다` (수동형 지양)
- **두음법칙** 적용: `이용(利用)`, `연락(連絡)`
- 불필요한 한자어·외래어 남용 금지

### 4-3. 제목 작성 요령

- 명사형으로 종결: `○○ 계획 수립`, `○○ 협조 요청`
- 내용을 대표하는 핵심어 포함
- 의문형·감탄형 사용 금지

### 4-4. 본문 첫 줄

본문은 두 가지 방식으로 시작합니다.

- **목적형**: `○○ 업무와 관련하여 다음과 같이 알려드립니다.`
- **근거형**: `「○○법」 제○조에 따라 ○○을 통보합니다.`

### 4-5. 붙임 표기

```
붙임  1. ○○○  1부.
      2. ○○○  1부.  끝.
```

- 마지막 붙임 항목 끝에 `끝.` 표기
- 붙임이 없을 경우 본문 마지막 줄 끝에 `끝.` 표기

## 5. 마크다운 출력 형식

기안문을 마크다운으로 출력할 때 아래 형식을 따릅니다.

```markdown
**수신:** [수신기관명] [수신자 직위·성명]
**(경유):** [경유기관명] (해당 없으면 이 행 삭제)

---

**제목:** [문서 제목]

1. [배경 또는 목적]
   [내용 서술]

2. [주요 내용]
   가. [세부 사항 1]
   나. [세부 사항 2]

3. [요청 또는 협조 사항]
   [내용 서술]

붙임  1. [첨부파일명]  1부.
      2. [첨부파일명]  1부.  끝.

---

[발신기관명] [대표자 직위]

담당: [부서명] [직위] [성명]
전화: [전화번호] / 팩스: [팩스번호]
공개구분: 공개
```

---

## 6. 자주 하는 실수

| 실수 유형 | 잘못된 예 | 올바른 예 |
|-----------|-----------|-----------|
| 날짜 형식 | `2024년 6월 26일` | `2024. 6. 26.` |
| 금액 표기 | `백만원` | `1,000,000원(일백만원)` |
| 문체 혼용 | `합니다 / 한다` | `합니다` 로 통일 |
| 제목 종결 | `○○을 요청합니다` | `○○ 요청` |
| 끝 표기 누락 | 본문 끝에 표기 없음 | 마지막 항목 뒤 `끝.` |
| 항목 번호 오류 | `1. → 1) → 가.` | `1. → 가. → 1)` |

---

## 7. 작성 체크리스트

- [ ] 수신자 정보가 정확한가
- [ ] 제목이 내용을 대표하는 명사형인가
- [ ] 항목 번호 체계가 올바른가 (1. → 가. → 1) → ...)
- [ ] 날짜 형식이 `년. 월. 일.` 형식인가
- [ ] 금액에 한글 병기가 되어 있는가
- [ ] 경어체(`~합니다`)가 일관되게 사용되었는가
- [ ] 본문 또는 마지막 붙임 항목에 `끝.` 표기가 있는가
- [ ] 담당자 연락처 및 공개구분이 기재되었는가
